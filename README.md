# 原神圣遗物数据导出工具
## 项目简介
本项目基于 Enka.Network 公开 API 开发，可自动获取原神玩家公开展柜的角色圣遗物数据，完成数据解析、圣遗物评分、有效词条统计与评级，并导出为美化格式的 Excel 文件。项目配套角色ID对照表自动更新工具，支持多镜像下载与本地缓存，可快速适配游戏新版本角色。

## 核心功能
1. 支持单个/多个原神 UID 批量导出圣遗物数据
2. 自动解析圣遗物套装、部位、主副词条、等级、稀有度等完整信息
3. 提供圣遗物评分、评级与总有效词条统计功能
4. 导出带样式美化的 Excel 文件，按角色自动分工作表展示
5. 配套角色ID映射表自动更新工具，支持多镜像下载与本地缓存
6. 网络异常时自动使用本地缓存，保障工具稳定性

## 文件说明
- `genshin_artifacts.py`：圣遗物数据导出主程序
- `update_avatar_map.py`：角色ID对照表自动更新工具
- 导出文件：以 UID+时间戳命名的 Excel 格式报表

## 运行环境
- Python 3.6 及以上版本
- 依赖库：requests、openpyxl
